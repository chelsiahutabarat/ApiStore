# API Store App

API Store adalah aplikasi React Native berbasis Expo yang menggunakan Fake Store API untuk menampilkan daftar produk beserta detailnya.

## API
FakeStore API (https://fakestoreapi.com/)

## Fitur
- Pull-to-Refresh
- Search / Filter produk
- Detail produk
- Empty state handling
